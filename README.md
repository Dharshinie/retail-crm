ShopPulse CRM

Features:

- User authentication (login/register/logout)

- Customer management (name, phone, loyalty points, total spent)

- Product management (name, price, stock with low stock alerts)

- Billing system (select customer, add multiple products, auto 
total calculation, store orders, reduce stock)

- Loyalty system (₹100 = 1 point, redeem points for discount)

- Credit system (buy now pay later, track pending payments and history)

- Dashboard & reports (daily sales, top customers, best-selling products)

- Smart suggestions (frequently bought together)
